LazyCode 是什么？
===================================  
LazyCode 一个懒人用的代码类库，包括的模块：网络请求，数据存储，UIKit ,Foundation的一些类目

PHNetWorkClient 功能 
----------------------------------- 
1:发送GET PSOT Delete PUT 请求
2：使用的URL缓存 可以设置缓存过期时间
3:上传文件，图片等带有进度
4:下载文件，图片等带有进度
5:控制并发数量根据当前网络。比如Wi-Fi 和手机网络情况下设置并发数。还可以设置仅仅Wi-Fi情况下来请求
6:设置请求超时时间
7:取消某个网络请求，取消所有的请求

Cache 功能 
----------------------------------- 
1:硬盘缓存
2:内存缓存
3:plist文件缓存
4:钥匙串缓存
5:自定义类的缓存

Foundation 功能 
----------------------------------- 
System 一些系统常用信息
PHSandBox 沙盒常用操作
ExtenSion  一些Foundation 类目

UIKit 功能 
----------------------------------- 
常用UIKit控件的扩展
